# Features

- Landing Page
    Hero section, features, tech tags, and community mission.

- Challenge Page
    Interactive bounty cards with neon glow, tilt hover animations, and particle background.

- Navigation Bar
    Persistent header with easy page switching.

- Reusable Components
    Modular structure for Hero, Features, TechTags, NavBar, and Layout for maintainability.

- Dummy Data Integration
    Features, tech tags, and challenges are all loaded from src/data for easy updates.

# Concept Behind the Design

The design demonstrates cyberpunk aesthetics and modern frontend interactivity:

- Engaging Interactions: Hover, tilt, and motion effects make the UI feel alive and interactive.

- Cyberpunk Futuristic Theme: Neon gradient glows, dark mode, and particle background evoke a high-tech underground dev scene.

- Minimal but Memorable: Clean layout with small details—animations, color accents, badges—create a strong impression.

- Community-Oriented Messaging: Features reflect the mission of Kracked Devs: share knowledge, empower devs, and have fun.

- Typography & Color Palette: Monospace fonts and neon cyan/pink/violet gradients highlight key content for a futuristic vibe.
